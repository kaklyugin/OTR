Для корректной работы программы с кирилице необходимо переключить кодировку консоли на "windows-1251".
Для этого :
1)Запустить консоль, щелкануть на заголовке окна , выбирать "Свойства".
2)В "Свойствах" на закладке "Шрифт" установить  шрифт "Lucida Console" , нажать "ОК".
3)Набрать в консоли "ChCp 1251", в ответ должен быть "Текущая кодовая страница: 1251". 

Для запуска программы ввести в консоли : 

java -jar "JavaPhoneBook.jar"
